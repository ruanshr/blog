# 前端性能之Chrome的Waterfall

浏览器根据HTML中外连资源出现的顺序，依次放入队列，然后根据优先级确定向服务获取资源的顺序。同优先级的资源根据HTML中出现的先后顺序来向服务器获取资源

瀑布中各项内容的含义：

排队：出现下面的情况时，浏览器会把当前请求放入队列中进行排队
1、有更高优先级的请求时
2、和目标服务器已经建立了6个TCP链接（最多6个，适用于HTTP/1.0 和 HTTP /1.1）
3、浏览器正在硬盘缓存上简单的分配空间
